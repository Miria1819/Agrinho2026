# 🌾 Agro Forte, Futuro Sustentável

Um projeto web interativo e responsivo desenvolvido para destacar a importância do equilíbrio entre a produção agrícola e a preservação do meio ambiente. A página apresenta informações cruciais sobre culturas agrícolas e inclui um minijogo educativo de ordenação.

---

## 🚀 Funcionalidades

* **Painel Inicial Animado:** Cabeçalho dinâmico com animações fluidas em CSS representando elementos da fazenda.
* **Ciclo de Vida dos Grãos:** Tabela informativa detalhando o tempo médio de produção e a melhor estação de cultivo para grãos populares (Soja, Milho, Trigo e Feijão).
* **Jogo "Separador Sustentável":** Um minijogo interativo que utiliza a API nativa de *Drag and Drop* (Arrastar e Soltar) do JavaScript, desafiando o usuário a categorizar insumos e alimentos (Verduras, Legumes e Esterco) em suas respectivas caixas ecológicas.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web puras (Vanilla), sem a necessidade de frameworks externos:

* **HTML5:** Estruturação semântica da página e dos elementos do jogo.
* **CSS3:** Estilização moderna utilizando Variáveis CSS (`:root`), Flexbox para alinhamento e `@keyframes` para as animações.
* **JavaScript (ES6):** Lógica de manipulação do DOM e implementação dos eventos de arrastar e soltar.
